# nanotodon
TUI/C99 Mastodon Client

# Depended Library
- cURL
- ncursesw
- json-c
- pthread

# How to build
```make```

# Options

- ```-mono```  
 - Draw as monochrome mode.

# Tested environments
- NetBSD/luna68k + mlterm
- NetBSD/x68k + mlterm
- NetBSD/sun3 + mlterm
- ArchLinux + xfce terminal
- WSL1 + mintty(wsl-terminal)
